FPGA Board specification: 

Zynq®-7000 All Programmable SoC XC7Z020-CLG484-1 
Memory
512 MB DDR3
256 Mb Spansion® Quad-SPI Flash
4 GB SD card
Onboard USB-JTAG Programming
10/100/1000 Ethernet
USB OTG 2.0 and USB-UART
PS & PL I/O expansion (FMC, Digilent Pmod™ compatible, XADC)
Multiple displays (1080p HDMI, 12-bit VGA, 128 x 32 OLED)
I2S Audio CODEC
